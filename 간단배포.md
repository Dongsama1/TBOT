# 🚀 간단 배포 실행 방법

## 방법 1: 배치 파일 실행 (가장 쉬움)

**`배포실행.bat`** 파일을 더블클릭하세요!

이 파일이 PowerShell 스크립트를 자동으로 실행합니다.

---

## 방법 2: PowerShell에서 직접 실행

1. 프로젝트 폴더에서 **우클릭** → **"터미널에서 열기"** 또는 **"PowerShell에서 열기"**

2. 다음 명령어 실행:
   ```powershell
   .\자동배포.ps1
   ```

3. 만약 실행 정책 오류가 나면:
   ```powershell
   Set-ExecutionPolicy -Scope Process -ExecutionPolicy Bypass
   .\자동배포.ps1
   ```

---

## 실행 중 안내

스크립트가 자동으로:
1. ✅ Git 확인 및 초기화
2. ✅ 파일 커밋
3. ✅ GitHub 저장소 연결 (처음만 URL 입력 필요)
4. ✅ 코드 업로드
5. ✅ Render.com 배포 안내

---

## 문제 해결

### "실행 정책 때문에 스크립트를 실행할 수 없습니다"
```powershell
Set-ExecutionPolicy -Scope Process -ExecutionPolicy Bypass
.\자동배포.ps1
```

### "Git을 찾을 수 없습니다"
- Git 설치 확인: https://git-scm.com/download/win
- PowerShell 재시작

### "GitHub 인증 실패"
- GitHub Desktop 사용 권장
- 또는 Personal Access Token 생성
