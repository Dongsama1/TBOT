===========================================
AI Smart Service - 실행 방법
===========================================

이 프로젝트는 React + Vite 기반이므로 Node.js가 필수입니다.

[방법 1] 자동 실행 (권장)
-------------------------
1. "실행방법.bat" 파일을 더블클릭하세요
2. Node.js가 설치되어 있지 않으면 설치 안내가 표시됩니다
3. Node.js 설치 후 다시 "실행방법.bat"를 실행하세요

[방법 2] 수동 실행
------------------
PowerShell 또는 명령 프롬프트에서:

1. 프로젝트 폴더로 이동:
   cd C:\Users\Dongsama\Desktop\ai-smart-service

2. 의존성 설치:
   npm install

3. 개발 서버 실행:
   npm run dev

4. 브라우저에서 접속:
   http://localhost:5173

===========================================
중요 사항
===========================================

❌ index.html 파일을 직접 열면 작동하지 않습니다!
✅ 반드시 "npm run dev" 명령어로 실행해야 합니다!

Node.js 설치:
- https://nodejs.org/
- LTS 버전 다운로드 및 설치
- 설치 후 PowerShell 재시작
